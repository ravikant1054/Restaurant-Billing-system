Project Overview

This project is a Smart Restaurant Ordering System developed using Python.
The system allows customers to order food items from a menu, modify their order, and automatically calculate the total bill including GST.

The program also stores customer order data in a CSV file and performs basic data analysis and prediction to estimate what item a customer is most likely to order next time.

This project demonstrates basic Python programming, data analysis, visualization, and simple prediction logic.

---

 Features

- Interactive restaurant menu system
- Customer order input with quantity
- Option to modify the order
- Automatic bill calculation with 18% GST
- Order visualization using Matplotlib
- Data storage in CSV file
- Order history tracking using Pandas
- Simple customer order prediction system

---

 Technologies Used

- Python
- Pandas
- Matplotlib
- Scikit-learn
- CSV Data Storage

---
 How the System Works

1. User starts the system by typing hello
2. Menu items are displayed
3. Customer selects items and quantity
4. System calculates the total bill with GST
5. Order summary graph is displayed
6. Order data is saved in Restaurant_Data.csv
7. Previous customer orders are analyzed
8. The system predicts the most likely item the customer may order next time

---
Project Structure

restaurant_order_system.py → Main Python program
Restaurant_Data.csv → Customer order history data

---

 How to Run the Project

1. Install required libraries

pip install pandas matplotlib scikit-learn

2. Run the Python program

python restaurant_order_system.py

3. Type hello to start ordering.
